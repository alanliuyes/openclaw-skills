# PPT Generator Skill

将用户讲稿一键生成乔布斯风极简科技感竖屏HTML演示稿。当用户需要生成PPT、演示文稿、Slides、幻灯片，或要求科技风/极简风/乔布斯风格的演示时触发此技能。输出为单个可直接运行的HTML文件。

## Usage

Use this skill when users need:
- PPT generation
- Presentation slides
- Tech-style presentations
- Minimalist design
- Steve Jobs style presentations

## Output

Single HTML file that can run directly

## Style

- Steve Jobs style
- Minimalist tech aesthetic
- Vertical screen optimized
