# PECO Loop - Infinite Oracle Core

A Python-based orchestration system for managing PECO (Perpetual Enhancement and Continuous Operation) workers.

## Features

- Proactive installation management
- Durable desire injection
- Human-in-the-loop operations
- Feishu integration

## Usage

```python
from peco_loop import PECOWorker

worker = PECOWorker()
worker.start()
```
