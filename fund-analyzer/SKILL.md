# Fund Analyzer Skill

提供基金净值查询、历史走势、基金筛选排行、持仓分析等功能，数据来源于支付宝基金页面。用于用户查询基金信息、分析基金表现、筛选优质基金等场景。

## Usage

Use this skill for:
- Fund net value queries
- Historical trend analysis
- Fund screening and ranking
- Portfolio analysis

## Data Source

支付宝基金页面

## Features

- Comprehensive fund data
- Performance analysis
- Ranking and screening tools
