# OpenClaw Skills

This repository contains all the OpenClaw skills for extending AI agent capabilities.

## Skills Overview

This collection includes 28+ skills covering:

- **Web & Search**: web-search, multi_search_engine, agent_browser, playwright
- **Document Processing**: pdf, word-docx, Excel-xlsx, ppt-generator
- **Financial Analysis**: stock_analysis, china-stock_analysis, fund-analyzer, eastmoney, market_predictor
- **Development**: github, frontend-design-3, skill_creator, remotion-best-practices
- **AI & Media**: Seedream-image-for-openclaw, seedance, canvas-design-2
- **Data & Knowledge**: ontology, session_memory, self_improving, self_improving_agent
- **Communication**: imap-smtp-email
- **Utilities**: weather, find_Skills, infinite_oracle

## Structure

Each skill is organized in its own directory with:
- `SKILL.md` - Main skill documentation
- `scripts/` - Supporting scripts (Python, JavaScript, Shell)
- Configuration files (`_meta.json`, `package.json`, etc.)

## Usage

Skills can be installed and used within the OpenClaw framework to extend AI capabilities.
