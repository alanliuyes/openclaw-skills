# 市场驱动因素参考

## 一、地缘政治因素（权重30%）

### 影响机制

地缘政治事件通过以下路径影响市场：

1. **避险情绪** → 黄金、美元上涨，股市下跌
2. **供给冲击** → 原油、大宗商品价格波动
3. **贸易影响** → 相关国家股市、汇率波动
4. **政策预期** → 央行政策调整预期变化

### 关键事件类型

| 事件类型 | 影响资产 | 持续时间 |
|---------|---------|---------|
| 军事冲突升级 | 黄金、原油、军工 | 1-4周 |
| 制裁措施 | 相关国家股市、货币 | 1-3月 |
| 贸易谈判 | 港股、A股、汇率 | 1-2周 |
| 选举结果 | 相关国家股市、债券 | 1-4周 |
| 恐怖袭击 | 避险资产短期上涨 | 1-5天 |

### 监测重点

- **中东地区**：霍尔木兹海峡、伊朗核问题、巴以冲突
- **东欧地区**：俄乌冲突进展
- **亚太地区**：朝鲜半岛、台海局势
- **中美关系**：经贸谈判、科技制裁

## 二、技术面因素（权重25%）

### 核心指标

| 指标 | 含义 | 判断标准 |
|------|------|---------|
| **趋势方向** | MA5/MA10/MA20 | 多头排列=看涨 |
| **支撑阻力** | 前期高点/低点 | 突破阻力=看涨 |
| **成交量** | 量价配合 | 放量上涨=看涨 |
| **换手率** | 活跃程度 | 3-5%为正常 |
| **RSI** | 超买超卖 | >70超买，<30超卖 |
| **MACD** | 趋势强度 | 金叉看涨，死叉看跌 |

### 趋势判断

```
强上涨：MA5 > MA10 > MA20，成交量放大，RSI 50-70
弱上涨：MA5 > MA10，但MA10 < MA20，成交量一般
震荡：均线交织，成交量萎缩
弱下跌：MA5 < MA10，但MA10 > MA20
强下跌：MA5 < MA10 < MA20，成交量放大
```

## 三、资金流向因素（权重25%）

### ETF资金流向

| 指标 | 含义 | 判断标准 |
|------|------|---------|
| **份额变化** | 增减份数 | 增加=资金流入 |
| **净流入/流出** | 当日资金 | 正值=流入 |
| **融资买入** | 杠杆资金 | 增加看涨 |
| **融券余额** | 做空力量 | 增加看跌 |

### 北向资金

- **北向净买入**：外资看好A股 → 利好
- **北向净卖出**：外资看空A股 → 利空
- **连续流入/流出**：趋势确认信号

### 主力资金

- **主力净流入**：大资金进场 → 利好
- **主力净流出**：大资金离场 → 利空
- **超大单动向**：机构资金方向

## 四、市场情绪因素（权重20%）

### 情绪指标

| 指标 | 来源 | 判断标准 |
|------|------|---------|
| **恐慌贪婪指数** | CNN | <20恐慌，>80贪婪 |
| **VIX波动率** | CBOE | >25市场恐慌 |
| **涨跌停比** | A股 | 涨停多=情绪高 |
| **换手率** | 交易所 | 过高=过热 |
| **舆情热度** | 新闻媒体 | 关注度高=活跃 |

### 情绪周期

```
极度恐慌 → 恐慌 → 正常 → 贪婪 → 极度贪婪
(底部区域)          (正常)        (风险区域)
```
