# Market Predictor V2 - 多资产市场预测分析（增强版）

基于实时数据的多资产市场预测分析工具，输出结构化概率表格和风险提示。

## 核心特性

- ✅ **全品种覆盖** - ETF、LOF、QDII、场外基金、股票、指数、大宗商品
- ✅ **实时数据** - 搜索过去3小时内的最新市场信息
- ✅ **宏观数据整合** - 自动搜索地缘政治、央行政策、重大事件
- ✅ **四因素分析** - 地缘政治(30%) + 技术面(25%) + 资金流向(25%) + 市场情绪(20%)
- ✅ **概率表格** - 即时/近期/中期/长期四个时间维度
- ✅ **溢价风险检测** - 自动识别LOF/QDII的场内溢价风险

## 工作流程

```
Step 1: 确认时间 → Step 2: 识别资产 → Step 3: 批量搜索 → Step 4: 宏观搜索 → Step 5: 数据分析 → Step 6: 概率预测 → Step 7: 风险提示
```

## Step 3: 批量搜索市场数据

使用 `scripts/search_market.py` 批量搜索资产数据：

```bash
# 设置时间范围（过去3小时）
FROM_TIME=$(python -c "import time; print(int(time.time()) - 10800)")

# 调用搜索脚本
python scripts/search_market.py --date "2026年3月29日" --from-time $FROM_TIME
```

### 关键数据提取

从搜索结果中提取：

1. **价格数据**：最新价格、涨跌幅、成交量

3. **重要事件**：溢价风险、停牌、分红、政策利好
4. **技术信号**：趋势、支撑阻力、换手率

## Step 5: 数据分析与溢价检测

### 资产状态摘要表

将搜索结果整理为结构化表格：

| 资产 | 代码 | 最新价格 | 涨跌幅 | 净值 | 溢价率 | 关键事件 |
|------|------|---------|--------|------|-------|---------|
| 原油LOF易方达 | 161129 | 2.389元 | - | 1.6368元 | **+46%** | 溢价风险，多次停牌 |
| 黄金ETF | 518880 | ... | +1.5% | - | - | 避险需求 |
| 纳斯达克ETF | 513100 | 1.708元 | -0.47% | - | - | 主力流出 |

### 溢价风险判断

**LOF/QDII溢价率计算**：
```
溢价率 = (场内价格 - 基金净值) / 基金净值 × 100%
```

| 溢价率 | 风险等级 | 建议 |
|-------|---------|------|
| < 2% | 低 | 正常交易 |
| 2-5% | 中 | 谨慎追高 |
| 5-10% | 高 | 不建议买入 |
| > 10% | 极高 | **强烈规避** |

## Step 6: 概率预测表

### 多因素分析框架

| 因素 | 权重 | 分析要点 |
|------|------|---------|
| **地缘政治** | 30% | 冲突升级/缓和、制裁、贸易政策 |
| **技术面** | 25% | 趋势方向、支撑阻力、成交量 |
| **资金流向** | 25% | ETF流入/流出、北向资金、融资 |
| **市场情绪** | 20% | 恐惧贪婪、新闻情绪、舆情热度 |

### 概率判断标准

| 确定性等级 | 概率范围 | 判断条件 |
|-----------|---------|---------|
| **高确定性** | 70%+ | 强趋势+重大事件确认+多因素共振 |
| **中等确定性** | 55-70% | 趋势明确但有扰动+主要因素主导 |
| **低确定性** | 50-55% | 方向不明+因素矛盾+等待催化剂 |

### 标准输出格式

```
| 资产 | 即时(1-3天) | 近期(1-2周) | 中期(1-3月) | 长期(6-12月) | 核心逻辑 |
|------|------------|------------|------------|-------------|---------|
| XXX | ↑ 75% / ↓ 25% | ↑ 65% / ↓ 35% | ↑ 55% / ↓ 45% | ↑ 50% / ↓ 50% | 简要说明 |
```

**概率格式**：`↑ XX% / ↓ YY%`（上涨/下跌概率，总和100%）

## Step 7: 风险提示与投资建议

### 必须包含的内容

1. **关键假设**：影响预测的核心前提（2-4条）
2. **敏感性分析**：假设变化对结果的影响
3. **数据局限性**：数据来源、时效性说明
4. **免责声明**：不构成投资建议

### 投资建议分类

| 策略方向 | 适用资产 | 判断标准 |
|---------|---------|---------|
| **避险配置** | 黄金、军工 | 地缘冲突升级+资金流入 |
| **政策受益** | 新能源、电力 | 政策支持+行业景气+资金流入 |
| **估值修复** | 中概互联、恒生科技 | 估值底部+催化事件+资金流入 |
| **防御稳健** | 现金流ETF、红利 | 高股息+低波动 |
| **谨慎观望** | 高溢价LOF/QDII | 溢价率>5% |
| **逢低布局** | 超跌行业ETF | 技术面调整+资金流入+中长期看好 |
