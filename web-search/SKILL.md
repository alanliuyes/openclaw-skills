# Web Search Skill

Use this skill when users need to search the web for information, find current content, look up news articles, search for images, or find videos.

## Usage

Simply use this skill when you need to search the web. The skill uses DuckDuckGo's search API to return results in clean, formatted output (text, markdown, or JSON).

## When to Use

- Researching topics
- Fact-checking
- Finding recent information
- Gathering web resources
- Looking for images or videos

## Output Format

The search results are formatted in clean, readable output that can be text, markdown, or JSON depending on your needs.
